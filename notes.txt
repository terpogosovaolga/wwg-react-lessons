Сайт был разработан специально для курса по дисциплине для дальнейшего размещения на React. Инструкция по установке React находится ниже. 
Верстка не представляет из себя никакой ценности и сделана в качестве учебного примера.
Все картинки целенаправленно импортированы ссылками на внешние источники - для уменьшения количества файлов на скачинвание. К тому же в процессе работы с React это покажется вам намного удобнее. В реальных проектах настоятельно рекомендую скачивать все изображения!
Использованы html+css, также взяты иконки с font-awesome. JS создавать смысла нет - все будем писать на React. 

Инструкция по установке react:
1. Установите node.js. 
2. В директории, в которой будет создана папка с проектом необходимо запустить терминал - например, в VS Code. Один из способов это сделать - открыть папку в VS Code, в верхнем меню найти пункт Terminal (Терминал) и нажать "New Terminal" (Новый терминал).
3. В открывшейся консоли пишем: npm create-react-app my-app, где my-app - это название будущего проекта. Утсановка будет достаточно долгой. 
4. У вас должна появиться директория, названная одноименно с вашим проектом. Теперь все команды терминала будут в ней. Можно либо открыть заново эту папку в VS Code, либо в терминале написать cd /my-app (вместо my-app - название папки с проектом).
5. Чтобы запустить проект, пишем: npm run start или npm start. Должен открыться браузер с приложением React (темная страница с логотипом компании).
